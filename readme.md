<h1 align="center">
    A Vocab Memorization Application
</h1>

Technologies used:

- Django

- React

- Redux

- Beautiful Soup (for scraping dictionaries)

Functionalities:

- Add/Remove vocab

- Undo/Redo

- Get meaning from Oxford and Cambridge dictionary (beautiful soup in backend)

- Authentication system

- Add profile picture
